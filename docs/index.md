# ConcursoFácil

O **ConcursoFácil** é uma plataforma de apoio ao estudo para concursos públicos.  
O sistema permite que o candidato organize seus estudos a partir de **editais específicos**, resolva questões por disciplina e acompanhe seu desempenho por meio de métricas e simulados.  

O projeto busca facilitar a preparação dos estudantes, oferecendo uma solução centralizada e intuitiva para gerenciar conteúdos, praticar questões e obter feedback sobre a evolução.  
Nosso objetivo é contribuir para que candidatos tenham **mais eficiência, foco e clareza** em sua jornada de aprovação.
